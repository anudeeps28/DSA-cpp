## Question 3
**Write a program to find the transpose of a matrix.**

**What is Transpose?**  
Transpose of a matrix is the process of swapping rows with columns.  

For a 2×3 matrix:
```
Matrix:
a11 a12 a13
a21 a22 a23

Transposed Matrix:
a11 a21
a12 a22
a13 a23
```
